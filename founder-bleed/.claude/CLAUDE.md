# Founder Bleed

A web app that helps founders reclaim time by analyzing their calendar, classifying work by delegation & discipline, delegating work to team members, generating hiring recommendations and planning their time more effectively.

## Tech Stack
- Next.js 14+ (App Router), TypeScript, Tailwind CSS, shadcn/ui
- Drizzle ORM with PostgreSQL (Neon)
- NextAuth.js v5 with Google OAuth
- Stripe for payments
- next-themes for dark/light mode

## Core Concepts
- 5-Tier Model: Unique, Founder, Senior, Junior, EA
- Core Loop: TRIAGE → DELEGATE → PLAN
- Solo founders see only 4 tiers (Founder hidden)
- Planning Score displayed as percentage
